GIt 的 单身时间  是  2005     GIT 和 SVN  都是 版本控制工具

1、 git数据备份 比svn 安全

2、 git速度 一般 比svn要快

3、工作流程 不一样  。 
		svn 的流程  ：每次 先从 服务器 上 更新代码， 修改完成后 上传到服务器上去。

		git 的流畅  ： 每次提交到服务器之间 需要 从 服务器更新代码，每次修改后，新增后 ， 提交到 本地库。
		
		git 本地多了 一个 可以  和服务器一样的 信息保存库。



SVN 和 Git 最大的区别 

SVN  是集中式的 版本控制 ，git是 分布式  版本控制工具

主要是 版本的  信息 存放的 位置不同。
